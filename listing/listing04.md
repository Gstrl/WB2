Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Происходит отправка значений от 0 до 9 в канал ch. Отправка значений в небуферизованный канал блокируется до тех пор, пока эти значения не будут приняты из канала.
Ниже приведен исправленный код, иначе возникает deadlock, for n := range ch
ждет до закрытия канала, а его не закрывают

```
```go
package main

func main() {
    ch := make(chan int)
    go func() {
        for i := 0; i < 10; i++ {
            ch <- i
        }
        close(ch) // Закрываем канал после отправки всех значений
    }()

    for n := range ch {
        println(n)
    }
}
```
